General instructions here:

1. Do not run the functions without uploading a file
2. Make sure that there are no empty rows at the beginning of your excel sheet
3. Make sure that the sheet contains a 'Particulars' column
4. The column to be used for remarks should exaclty be named as "Particulars"
5. The column to be used for totals should exaclty be named as "WITHDRAWALS" and "DEPOSITS"
6. The uploaded file should not be empty
7. The uploaded file's format should be ".xlsx" or ".xls"
